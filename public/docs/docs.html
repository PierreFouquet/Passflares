How Passflares Keeps You Safe

  • Client-side encryption — Your vault data is encrypted with AES-256-GCM using a key derived from your master password before it ever leaves your browser.
  • Zero-knowledge server — The server stores only encrypted blobs. It cannot read your passwords.
  • Scrypt password hashing — Your master password is hashed with Scrypt on the server for authentication, keeping the derivation key separate from the auth credential.
  • Short-lived sessions — You are automatically logged out after 5 minutes of inactivity.
  • Audit logging — Every login, vault access, and account change is logged for your security.
  • Rate limiting — Failed login attempts are rate-limited to prevent brute-force attacks.

Quick Start

  1. Create an account with a strong master password.
  2. Create your first vault from the dashboard.
  3. Add password entries to the vault.
  4. Click Save Vault Data after making changes.
